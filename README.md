# Sports Analytics: MLB Catcher Defensive Contributions

## Project Overview
Analysis of MLB catcher framing contributions using 2025 Statcast data. 
Built for 36-460/660 Sports Analytics at CMU.

## Setup
1. Clone the repo
2. Run the data pull section of `Corliss_Catcher_Framing.qmd` to generate local data files
3. Cached `.rds` files are gitignored and must be generated locally

## Data Sources
- Baseball Savant Statcast: https://baseballsavant.mlb.com
- MLB Glossary (framing): https://www.mlb.com/glossary/statcast/catcher-framing
- 2026 Official Baseball Rules: https://mktg.mlbstatic.com/mlb/official-information/2026-official-baseball-rules.pdf
