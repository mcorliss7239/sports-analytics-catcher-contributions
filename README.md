# Sports Analytics: MLB Catcher Defensive Contributions

## Project Overview
Analysis of MLB catcher defensive contributions across three dimensions:
pitch framing, pitch blocking, and count specific framing analysis using 2025 
Statcast data. Built for 36-460/660 Sports Analytics at CMU.

## Setup
1. Clone the repo
2. Each analysis file must be run locally to generate cached .rds files 
   before rendering report.qmd
3. Large data files (statcast_2025_raw.rds) are gitignored and must be 
   generated by running the data pull section of the relevant analysis file
4. Plot .rds files are tracked in the repo and will be available after pulling

## Files
- report.qmd — final report
- Corliss_Catcher_Framing.qmd — Morgan's framing analysis
- blocking_analysis.qmd — Arthur's blocking analysis
- Joel_Framing_Heterogeneity.qmd — Joel's count-specific framing analysis

- Baseball Savant Statcast: https://baseballsavant.mlb.com
